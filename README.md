ICHgram — Full-Stack Social Media App

Frontend
React client:
https://github.com/JuliiaJuliia/project-ichgram

Backend
Node.js + Express API:
https://github.com/JuliiaJuliia/project-ichgramBack

Features
• JWT Authentication (Register/Login)
• User profiles
• Create posts
• Like & comment system
• Follow / Unfollow users
• Stories
• Search users
• Notifications

Tech Stack

Frontend:
• React
• React Router
• Axios
• CSS

Backend:
• Node.js
• Express
• MongoDB
• Mongoose
• JWT Authentication

2. Install dependencies

Frontend:
cd project-ichgram
npm install

Backend:
cd project-ichgramBack
npm install

3. Environment variables

Create .env file inside backend folder and add:

PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key

4. Run project

Backend:
npm run dev

Frontend:
npm start

Frontend runs on:
http://localhost:3000

Backend runs on:
http://localhost:5000
